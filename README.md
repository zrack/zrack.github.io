# Wall of Maybe static site

This folder is ready to publish with GitHub Pages. Upload `index.html` and `og.png` to the root of the publishing branch.

Before publishing, replace the six project `href="#"` values in `index.html` with the live application URLs. Also replace the placeholder GitHub and LinkedIn links and confirm `hello@notether.com`.

If you use a custom domain, add a `CNAME` file containing the final hostname after DNS is configured.
